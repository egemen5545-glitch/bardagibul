# Bardağı Bul

HTML, CSS ve JavaScript ile hazırlanmış mobil uyumlu bardak-top oyunu.

## Dosya Yapısı

```text
bardagi-bul-proje/
├── index.html
├── style.css
├── script.js
├── assets/
│   ├── images/
│   └── sounds/
├── ORIGINAL_bardagi-bul.html
├── .gitignore
└── README.md
```

## VS Code'da Çalıştırma

1. Bu klasörü VS Code ile aç.
2. VS Code Extensions bölümünden **Live Server** eklentisini kur.
3. `index.html` dosyasına sağ tıkla.
4. **Open with Live Server** seçeneğine bas.

## GitHub'a Yükleme

VS Code terminalinde şu komutları çalıştır:

```bash
git init
git add .
git commit -m "İlk düzenli Bardağı Bul sürümü"
```

Sonra GitHub'da yeni repo oluşturup GitHub'ın verdiği `remote` ve `push` komutlarını çalıştır.

## Not

`ORIGINAL_bardagi-bul.html` dosyası, Claude'un ürettiği tek dosyalık orijinal sürümdür. Düzenli proje için ana dosyalar:

- `index.html`
- `style.css`
- `script.js`
